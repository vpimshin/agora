02 · The solution

A council of five agents.
One learning contract.

Not a chatbot. Not a course catalog. A coordinated multi-agent system, each role auditable, each output verifiable.

03 · How it works

Goal → Contract → Practice → Proof.

A four-step loop. Every step produces an artefact you can audit, replay, or revoke.

🎯
Step 1
Goal capture

Pedagogy paraphrases. Verifier insists on objectivity. Curator scopes. Together they negotiate a contract with the learner.

→ rubric.json
🗺️
Step 2
Living map

Curator builds a personalized knowledge graph. Drops 14× more results than it keeps. Every concept tied to a real artefact.

→ knowledge_graph.ttl
📅
Step 3
Daily practice

Companion plans 45 min/day, energy-aware. Empathy detects struggle and shrinks the task — never the person.

→ session.log
🎖️
Step 4
Verifiable proof

Open Badges 3.0 + W3C VC. Signed, portable, revocable. The learner owns the credential — not the platform.

→ credential.jsonld
🔒 Private by default — user data never trains models
🧠 Auditable reasoning — every agent step is logged
⚖️ Anti-Goodhart — Verifier rejects gameable metrics

05 · Lessons learned

What was easy.
What burned us.

✅
What flowed
The spec did the heavy lifting
  • Multi-agent split — once roles were named, the system designed itself.
  • Empathy as a UI palette — visualizing emotional state as color was instantly readable.
  • OB 3.0 / W3C VC — open standards meant zero vendor lock-in & one afternoon of integration design.
  • Glossary first — a single doc killed 80% of "wait, what do you mean by…" arguments.
🔥
What burned
Agents like to disagree
  • Agent conflicts — Pedagogy and Empathy fought over pace. Solved with an explicit arbitration ADR.
  • "Verifiable" is hard — turning "feel confident" into a falsifiable rubric is the real product, not a feature.
  • Anti-Goodhart guardrails — every metric we added invited a way to game it. We rewrote the token economy three times.
  • Scope discipline — the spec is so rich the temptation to "just code one more thing" is constant. We shipped a click-through, not a backend, on purpose.
